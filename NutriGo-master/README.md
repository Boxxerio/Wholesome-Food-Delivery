NutriGo - Healthy Meal Delivery Platform
NutriGo is a Spring Boot-based backend for a healthy food delivery platform focused on weight management. It allows users to personalize meal choices, track nutritional information, and place orders through a seamless authentication and filtering system.

Features:
User Authentication: Users can sign up and log in using their phone number or email with OTP verification.

Meal Filters: Filter meals based on:

Weight goals: Weight Loss, Weight Gain, Weight Management

Meal Types: Breakfast, Lunch, Dinner, Snack

Personal Preferences: Vegan, Vegetarian, Diabetic-friendly, PCOD-friendly, etc.

Meal Personalization: Users can select and personalize meals based on detailed nutritional specifications like calories, protein, carbs, and fat.

Order Placement: Users can place orders, which are then sent to nearby partner kitchens for preparation and delivery.

Technologies Used:
Backend: Spring Boot

Database: MySQL

Tools: Postman (for API testing)

Version Control: Git, GitHub
